# faircamp-feedstock

This repository contains the conda feedstock for [faircamp](https://codeberg.org/simonrepp/faircamp), a static site generator for audio producers.

## About faircamp

Faircamp is a static site generator specifically designed for audio producers. It creates beautiful, responsive websites for your music releases with minimal effort.

## Updating faircamp

To update the version of faircamp in this feedstock:

1. Update the version number in `recipe/meta.yaml`
2. Update the SHA256 hash in `recipe/meta.yaml`
3. Submit a pull request

## Building the package

To build the package:

```bash
conda build .
```
